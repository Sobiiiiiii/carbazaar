# 🚗 Car Bazar

A full-stack online vehicle marketplace developed using PHP, MySQL, HTML5, CSS3, JavaScript, and Bootstrap 5. The platform allows users to buy and sell vehicles through a simple and user-friendly interface with secure authentication and an admin dashboard.

---

## 📌 Features

- 🔐 User Registration & Login
- 🚗 Add, Edit & Delete Vehicle Listings
- 🔍 Search & Filter Vehicles
- 👤 User Profile Management
- 🛠️ Admin Dashboard
- 📱 Responsive Design
- 🗄️ MySQL Database Integration

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Sobiiiiiii/carbazaar.git
```

2. Move the project to the XAMPP `htdocs` folder.

3. Start **Apache** and **MySQL** in XAMPP.

4. Import the SQL database into phpMyAdmin.

5. Open your browser:

```
http://localhost/carbazaar
```
